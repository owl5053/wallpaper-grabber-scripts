# wallpaper-grabber-scripts

A collection of cross-platform scripts for automatically grabbing and updating desktop wallpapers from various online sources (Bing, 35photo, and more).

## 📂 Repository Structure

```
/windows - Batch and vbs scripts for Windows
/linuxmac - Bash scripts for Linux and macOS
```

## 🚀 Usage

1. Choose the script for your operating system from the appropriate folder
2. Add the selected script to your system's task scheduler:
   - **Windows**: Task Scheduler
   - **Linux**: crontab
   - **macOS**: launchd or crontab
3. The script will automatically fetch new wallpapers and update your desktop background
